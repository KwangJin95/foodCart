# 쇼핑몰(식품) [FOOD CART]


* **이름** : foodCart (푸드카트)
* **기간** : 2주
* **인원** : 풀스택 4명
* **소개** : MVC model-1 패턴에서 Servlet과 JSP를 사용한 개발 및 웹페이지 구현을 목적으로 마켓컬리를 모방하여 만든 쇼핑몰 사이트입니다.
